B站技术胖专栏程序员软技能

一，真相

技术为王？假话。

技术能够满足基本需求，听话更重要！

程序员创业容易？看技术？

交际管理协调能力，社会认知

程序员靠技术就可以安枕无忧了？

拿健康换钱，编程只是赚钱的工具，没到顶峰别幻想一辈子靠技术生存

编程作为兴趣爱好推荐嘛？

不推荐，这门技术更新迭代很快，需要投入大量的时间精力。

喜欢学习新技术而不愿意去总结优化旧的东西

先把已经学的东西消化好了，再去学习新的东西。

这个不只是编程，我学东西也喜欢急功近利，追求学习新的东西的新鲜感，满足感，虚荣感

生活中也是，喜欢学习新的东西而不愿意优化回顾已经了解的东西

代码能跑就行，搞什么重构费力不讨好好容易出事，还不给钱/完成自己的任务就行了，不要多管闲事

屎山代码带来的 长期成本，人类惰性

技术负债，短视，长远考虑，态度，

原因

  1. 没意识不重视短期付出和长期收获。
  2. 学习不复习

我的生活中像这种技术负债的情况还挺多的，不联系打字，拖着英语不去学习什么的，最后先问题越滚越大，直到无法解决

  1. 前端三剑客的Demo实践：Mecenlee/50projects50days: (前端三件套Demo)50+ mini web projects using HTML, CSS & JS (github.com)
  2. 前端性价比低的技术罗列：JQuery JSP PHP Angular JS（前端框架 ）低代码平台 Flutter（对于前端RN）过时，过于简单
