Android 学习路线规划

一，语言学习

1.1 Java SE初级入门

  1. 书籍：《Java核心技术》

    其中界面编程可以弱化

  2. 网课：韩顺平 Java 8补充

1.2 Kotlin

  1. 我的Android入门视频中Kotlin部分/《第一行代码 第三版》中Kotlin部分
  2. 官网
  3. 进阶：《Kotlin编程权威指南》

1.3 Andoid 初级入门

  1. 我在做的Android入门教程
  2. 《第一行代码 第三版》（这个架构的很好但是19年的现在2022有很多地方有问题，OS：我就挖出来上百个。人都麻了）

1.4 Java SE 进阶

  1. 《Java 编程思想》
  2. 《Effective Java》
  3. 刷牛客的 Java 题目：https://www.nowcoder.com/intelligentTest
  4. 用 Java 刷力扣

1.5 Android 进阶

  1. 《Android开发艺术探索》
  2. 《Android源码设计模式解析与实战》
  3. 《群英》

1.6 Android实战

学习主流 Android 开源库

1.网络请求（Retrofit2）

2.图片加载（Glide／Fresco)

3.事件总线（EventBus/Otto）

4.异步操作（RxJava2)

5.依赖注入（ButterKnife/Dagger2）

数据库操作，图片选择器，大图查看器

注意：1.5&1.6同时进行

1.7 设计模式

二，计算机基础

2.1 数据结构和算法

  1. 入门：《小灰的漫画算法》/尚硅谷韩顺平Java算法
  2. 就业刷题：《代码随想录》/《剑指offer》
  3. 题库：力扣
  4. 进阶：《算法4th》

2.2 计算机网络

  1. 《图解HTTP》/《图解TCP/IP》
  2. 网课：湖科大PPT/哈工大
  3. 进阶：《计算机网络自顶向下》
  4. 尚有余力：《TCP/IP详解》

2.3 操作系统

  1. 入门：王道考研/《操作系统导论》
  2. 进阶：哈工大李志军操作系统/《深入理解操作系统》/《操作系统:精髓与设计原理》

三，高级

3.1 Java 底层

《深入理解Java虚拟机》

3.2 Android 工具

《Android群英传——神兵利器》

3.3 Andriod 底层

《深入理解 Android 虚拟机 ART》

3.4 跨端

Flutter/RN/Web前端

四，其他

4.1 Linux

《鸟哥的私房菜》

4.2 数据库

  1. 哈工大数据库
  2. 尚硅谷 Redis6
  3. 《MySQL必知必会》
  4. 《从根儿上理解mysql》
  5. 《MySQL技术内幕：InnoDB存储引擎》 （选）
  6. 《Redis设计与实现》

4.3 服务器端

  1. Java SE 韩顺平
  2. Java Web 尚硅谷最新版
  3. Java EE

五，方法

下面说的是大三上学期之前的

5.1 实践型学科

（此方法适用于c、c++、java、sql、linux等需要敲代码的课程）

把书上 经典代码敲打1-2遍以上（小片段用 Sublime），此方法也适用于《数据结构》，因为课本很多伪代码，你看着会，但是你实际运用过程中，漏洞百出，完全可以自己全部实现（刚开始如果不熟悉，可以去CSDN网站看别人怎么实现的，然后抄一遍，等你懂了后，再删了后，不看别人的代码，重新书写一遍），这一切都需要你的不懈努力+兴趣，每天坚持几个小时，你会发现一个月后，你就产生了质变。往回看这些以前头疼的代码，全部都迎刃而解。

5.2 抽象型学科

这个适合（计网，计组，操作系统）

我当时就是买来王道408，老师讲一节，我就会做一节这节课课后习题，然后在错误中查缺补漏，或者直接去牛客上刷题也行

5.3 其他

如果你性格孤僻，多刷leetcode、多参加个人比赛也可以，也可以去卡哥刷题网站刷，也可以去杭电acm网站刷题。 课下可以做做自己的个人网站（CSDN、github、自己的博客）

有了上述一定基础后，你大二就可以去做一些自己的小项目了（比如最经典的图书管理系统、爬虫、web项目、树莓派、飞机大战小游戏、仿QQ软件、QQ音乐播放器）
