【Android学习路线03】计算机基础

计算机基础就是计算机考研 408 的那 4 门课。

  1. 数据结构和算法(算法)
  2. 计算机网络(计网)
  3. 操作系统
  4. 计算机组成原理(计组)

咱们 Android 找工作的话，学前面三个就中。

算法

这个是最重要的。

基本头部企业要不要你，很大程度上就看你的算法能力。

这门课也是最难的，有数学的味道，比较费脑子。

我建议大家这门课放多点时间，别给自己太大压力。

然后早点开始学。

学算法分为两个部分：

  1. 入门
  2. 刷题

入门我推荐：韩顺平在B站讲的算法课。

他是用 Java 讲的，虽然我们写 Android 用的是 Kotlin，但是再找遍全网和问过很多 Android 前辈，结合目前的一个大情况。

我强烈建议所有学 Android 的小朋友，用 Java 刷算法。

入门学习自愿就是 老韩的这个，他的这套免费算法教程怎么使用。有哪些一定要注意的点，里面有很多学问的。我准备再单独写一期来细细展开说说。大家记得催我，不然又懒的鸽了。

然后学完了韩顺平的这个Java算法课之后，就可以去刷题了。

这个刷题也有讲究。

我给你规划的是这么个三步走：

  1. 《代码随想录》
  2. 《剑指Offer》
  3. LeetcodeHot200

啥意思呢？

就是按这个顺序是求职算法刷题最优解。

你给我保证按照我的给你这安排来的，把这三都每个刷了3遍以上。

吃透，基本面试算法，都能轻松cover。

记住啊，上面的就是顺序，吃透一步才能进入下一步。

别贪。

计网

这个第二学，因为 面试后端 他问操作系统多一点。

但面试 Web前端和移动端(Android/IOS)他问网络的更多。

计网对我们大前端也很重要。

所以第二个学哈。

这个入门推荐：B站湖科大教书匠的那个PPT讲解视频

顶级，我怎么跟你说。

别急，慢慢看，细细的看，我推的东西我都是走过的，是真心宝藏。

入门一般看视频快点，你要喜欢看书入门。我推荐：

《图解HTTP》《图解TCP/IP》

还是推荐看视频。

至于啥计网进阶的：

《计算机网络自顶向下》(你们学校要是有计网这门课，估计指定教材就是这个)

《TCP/IP详解》

啥的，不推荐啊，统统不推荐。

为什么？

初级程序员就看那个视频，然后背八股就中了。

别听那些大佬给你贩卖焦虑，别给自己太大压力。

操作系统

Android 不就是个操作系统。

你学完这个再去看Android的一些东西，你会觉得很透彻。

就之前上课的时候觉得很别扭啊，想不明白啊的那些地方。

其实就是知识逻辑链条少了一环。

你学完这个就可以补上很多了。

这个推荐入门视频就是：

B站王道考研的操作系统。

虽然人家是录得考研得，但是很实用。

还是不推荐进阶，找到工作干了两三年再来考虑进阶吧。

总结

学习顺序安排：

韩顺平Java算法->湖科大教书匠PPT计网->王道操作系统

你在算法上完了就可以去学计网，同时进行算法刷题。

就开始学算法：全天都看那个韩得视频

看完了就：半天刷算法题目，半天推进计网。

记住每天都要刷算法题。这个在你找到工作之前都要坚持。

这几年得学长拿到 30w Offer的，基本都走到的leetcode200。

累计刷了500多力扣。

你最好在大四之前，把《代码随想录》过两遍。

不然容易挂算法哪。

辣个计算机组成原理，面试其实不考。你要是有空，或者有兴趣，可以去看看。

看完当不成程序员做开发还能开个店修电脑(doge)

好了，我是火山君。

一个聚焦于 Android 开发入门的老哥。
