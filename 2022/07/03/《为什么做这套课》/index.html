为什么做这套课

发表于2022-07-03|更新于2023-03-20|生活IP
|阅读量:

《为什么做这套课》

大家好，今天是第二部分：为什么会专门做这么一套安卓开发入门课。

我把原因总结为以下两点：

  1. 市面上奇缺；
  2. 看不惯某线下安卓培训小老板的吃相。

市面上奇缺

大家有在网上搜安卓开发的相关学习资源嘛？

什么感觉？

少

还有~

旧。

我觉得“百废待兴”去形容现在的安卓开发学习资源还挺贴切的。

事实是全网几乎已经没有人再做安卓开发相关的教程和课了。培训机构也早早的就砍掉了相关业务。

为什么？

因为吃力不讨好，不挣钱。

所以就导致市面上流通的安卓学习资料，Bug解答还是5年前，8年前的圣遗物。

等你兴致冲冲的看了几集发现一点都对不上，这里错哪里错的。直接一波让你心态爆炸。

那

2023年还想学安卓开发容易入门嘛？是不是直接从入门到放弃了直接？

基本是这样，除非你自学能力很强，试错能力时间和机会都有。否则贫乏的安卓学习资源就是你学安卓的第一道难关。

知乎上的资源推荐，B站上的入门视频，当当上的相关书，CSDN公众号上的博客文章，还活着的搞安卓的博主……

我替大家全探过路了，结果是

还真没有。

一个留给安卓开发初学者入门的资源都没有，不是太旧了不适用了，就是错误百出。

那种感觉就是

找一个入门资源，学着学着发现2年前过期不用了

看一套教程，看着看着发现又现在的开发环境和那时候对不上了

出了问题，找遍全网回答都是5年前的前辈写的。

……

没有任何合适的学习资源

是我想做这套安卓入门课的第一个原因。

第二个是我再学校遇到的一个搞安卓培训的小老板。

当时发现网上实在没有什么好的自学资源，这个人又来到我们学校向我们专业开安卓开发培训班 。

收费 9800

我看到学院的领导都上去给他“背书”，加着当初的火山君还是个单纯的憨憨。

信了。

借好钱就报了。

到后面就发现不对劲了，全TM是营销套路。

安卓培训之前，先来个什么前端训练营。收你 4900。

近 5000 块就教了个 HTML CSS。

最大头的 JS 来不及了直接结课最后一天带完。

然后立马又招一次生，说上次的安卓开发班人头不够。

我虽然被坑了5000了，那个时候其实知道了。但是抗不过沉没成本太大，又跟了。

这次交了 9800 上了近40多天。没了，结课。

这次的安卓开发，带你安卓开发入个门，就结束了。

然后又马不停蹄的向我们下一届的学弟学妹去招生，开新的一期培训班。

又是全专业都强制去听他的卖课宣讲，还是领导上去“背书”，还是那套话术……

所以 总结一下。

教个 HTML,CSS，带一下 JS。收你 4800。

带你安卓简单入个门收你 9800。

已经不是贵不贵的问题了，这是明明可以直接抢，还要带你上几节课。

太温柔，我真的

哭死。

可能有些朋友如果不了解市场不知道这是什么概念。

举个例子。

全网讲 Java 最好的韩顺平老师的全套vip Java就业课

（质量在IT培训里面顶尖了，内容也是巨丰富从小白入门到你就业）

7000

然后类比到这个B这里。

本来值50（甚至免费）的前端那点东西，卖近5000。

值最多980的课他线下自己复述一遍，直接9800。

痛定思痛，我就决心一定要做出一套良心的安卓开发入门课。

让当初像火山君这种大冤种能少被割点韭菜，让各位后来人入门安卓的时候能

省时省力省钱。
