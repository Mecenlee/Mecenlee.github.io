Android 简介

一，Android 系统架构

  • Linux 内核层：底层驱动
  • 系统运行库层：C/C++实现的特性和 Android 运行时库（核心库，Dalvik -> ART虚拟机<Android 5>）
  • 应用框架：API
  • 应用

二，Android 版本发布

发布时间 Android 版本 备注
2008/09 Android 1.0 最早版本
Android 2.x 占领市场
2011/02 Android 3.0 专为平板设计(失败)
2011/10 Android 4.0 平板手机一视同仁
2014 Android 5.0 Dalvik -> ART | Material Design | 穿戴，汽车，电视Android 系统 （主要开发系统）
Android 10.0 目前智能手机的系统
Android 13.0 最新

三，Android 应用开发特色

  1. 组件：四大组件（Activity；Service；BroadcastReciver；ContentProvider），Jetpack
  2. UI：系统控件 布局 Material Design
  3. SQLite 数据库：轻量级，运算速度极快，嵌入式关系数据库。支持直接 SQL 语句 也支持 Android 封装好的 API
  4. 多媒体/网络

四，开发工具

  • JDK ：Java 开发工具包
  • Android SDK：Android 开发工具包
  • Sublime：写小片段代码的“记事本”
  • IDEA：Java/Kotlin 语言IDE
  • Android Studio ：专门开发 Android 的 IDE

五，开发语言

  1. Java ：老 Android 都是用这个开发的
  2. Kotlin：新 Android 官方指定开发语言（未来趋势）
  3. Jetpack Compose：新的搭积木
  4. Flutter：Android 跨端最火的路子 （语言：dart）
  5. RN（React Nactive）：Android 跨端的另一个路子，有不少公司用这个。（语言：Javascript）
